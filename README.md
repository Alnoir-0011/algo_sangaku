# [アルゴ算額](https://www.algosangaku.com)
[https://www.algosangaku.com](https://www.algosangaku.com)

## サービス概要
アルゴリズムの問題を作成し神社に算額として奉納、他ユーザーの作成した算額を解答することのできるサービスです

## ターゲットユーザー
- アルゴリズムの学習を行っているエンジニア、プログラミングスクール生、独学の方
